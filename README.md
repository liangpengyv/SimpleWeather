## 应用介绍

♤ 明亮的色彩和简洁的线条，传递您最需要的信息

♤ 一键分享天气截图给朋友，关心身边的人

♤ 支持自定义主题背景，多彩风格由您掌握

## 权限获取说明：

- 获取网络访问权限是为了从网络获取天气信息
- 获取SD卡读写权限是为了保存天气分享截图
- 获取访问设备电话功能是为了记录设备标识用以手机崩溃信息
- 查看WLAN网络状态是为了后台自动更新天气
- 查看网络连接状态是为了后台自动更新天气
- 开机启动是为了保证桌面小部件时间天气正常更新
